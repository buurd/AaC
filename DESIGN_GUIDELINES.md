# Design Guidelines

This document defines the visual style guide for the Webshop and Product Management System to ensure consistency and a modern look and feel.

## 1. Colors

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Blue** | `#007BFF` | Primary buttons, links, active states |
| **Secondary Grey** | `#6C757D` | Secondary buttons, borders, inactive text |
| **Success Green** | `#28A745` | Success messages, confirm actions |
| **Danger Red** | `#DC3545` | Error messages, delete actions |
| **Background** | `#F8F9FA` | Page background |
| **Surface** | `#FFFFFF` | Cards, modals, content areas |
| **Text Dark** | `#343A40` | Primary text |
| **Text Light** | `#6C757D` | Secondary text |

## 2. Typography

*   **Font Family**: `Arial`, `Helvetica`, sans-serif
*   **Headings**: Bold, Dark Text
    *   H1: 24px
    *   H2: 20px
    *   H3: 16px
*   **Body Text**: 14px, Line height 1.5

## 3. Components

### Buttons
*   **Padding**: `10px 20px`
*   **Border Radius**: `4px`
*   **Border**: None
*   **Cursor**: Pointer
*   **Primary Button**: Background `Primary Blue`, Text `White`
*   **Secondary Button**: Background `Secondary Grey`, Text `White`

### Tables
*   **Width**: 100%
*   **Border Collapse**: Collapse
*   **Header**: Background `Primary Blue`, Text `White`, Bold
*   **Rows**: Padding `12px`, Border Bottom `1px solid #DEE2E6`
*   **Zebra Striping**: Even rows background `#F2F2F2`

### Forms
*   **Inputs**: Padding `8px`, Border `1px solid #CED4DA`, Radius `4px`
*   **Labels**: Bold, Margin Bottom `5px`, Display Block

## 4. Layout
*   **Container**: Max-width `1200px`, Centered, Padding `20px`
*   **Spacing**: Use multiples of 8px (8px, 16px, 24px, 32px)
